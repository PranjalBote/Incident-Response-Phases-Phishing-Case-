# Incident Response Phases (Phishing Case)

## 1. Preparation
- Security awareness training
- Email security tools configured

## 2. Identification
- User reports suspicious email
- SOC detects malicious domain

## 3. Containment
- Disable compromised account
- Block malicious domain/IP

## 4. Eradication
- Remove malware (if present)
- Reset passwords

## 5. Recovery
- Restore affected systems
- Monitor account activity

## 6. Lessons Learned
- Improve email filtering
- Conduct awareness training
